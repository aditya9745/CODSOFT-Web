Netflix Viewer Analytics and Prediction

Situation

Netflix generates an immense amount of data, including user preferences, viewing times, search histories, and ratings. However, turning this vast pool of data into actionable insights for content recommendations and viewership predictions can be a considerable challenge, especially due to the size and complexity of the data.

Task

As the primary data analyst on this project, my aim was to construct a robust analytics pipeline using Python and Apache Spark, followed by an exploratory data analysis using SQL and a predictive model using linear regression. The goal was to develop a scalable process to handle large data volumes, clean, transform and load this data into a suitable format for a data warehouse, perform EDA, and ultimately create a prediction model..

Action

The initial phase involved assessing the data from various sources, including Netflix's API and social media platforms. I then designed a suitable data model and used Python and Apache Spark to extract, clean, transform, and load the data into a data warehouse
